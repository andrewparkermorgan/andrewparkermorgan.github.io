Overview

This page represents an effort to aggregate all existing whole-genome sequence data from wild house mice (Mus musculus) in a single location. For now I have dubbed it, unofficially, the Wild Mouse Genomes Project, although this is not an entity recognized by any institution or funding agency. The data linked here is either available in public repositories, or submission to such repositories is in process. The underlying biological specimens have been described in published literature. Except where otherwise indicated, the maintaner of this website does not claim credit for either the sequence data or the underlying biological specimen, and users of this data are encouraged to contact the data generators with any questions regarding provenance, reanalysis or reuse.

Methods

Raw paired-end sequence reads were dowloaded from the European Nucleotide Archive and aligned against the most recent version of the mouse reference genome, the GRCm39/mm39 assembly. The reference sequence was obtained from UCSC, so chromosome and contig names follow their nomenclature (chr1, ... chrM). Because the pseudoautosomal region (PAR) is the only region of the genome represented as diploid rather than haploid in the reference, one copy was masked (sequence forced to N) to avoid read-depth artefacts between sexes. Reads were aligned with bwa mem with default parameters, and putative optical/technical duplicates marked with samblaster in a streaming fashion. When applicable, alignments from multiple lanes per individual were merged with Picard. The resulting alignments were indexed with samtools in the usual fashion.

Sample sex and preliminary taxonomic assignment in the table below are based on a combination of information provided in the original publications and preliminary SNV calls in coding regions.

Full pipeline to be available on Github soon.

Alignments are available in *.bam format via Amazon Web Services in the public S3 bucket wild-mouse-genomes. No guarantee is made as to how long these files will be available as there is no funding stream to cover storage costs.

Samples

The table below lists samples currently available as part of the Wild Mouse Genomes Project. Sample IDs are those assigned by the original authors. Taxon assignments should be considered preliminary at best -- many of these individuals are inter-subspecies hybrids but are labelled with their "modal" ancestry. Location metadata is taken from the original publications. Countries are denoted by ISO 3166 2-letter code. "Region" and "site" are meant to indicate increasingly precise geographic place names. "Locale" is a 3-letter code, assinged by the author of this page, as a convenience for data aggregation. NB: None of the geographic labels here necessarily correspond to a "population" in the ecological or genetic sense.

"Accessions" link to the European Nucleotide Archive, which also mirrors the NCBI Short Read Archive (USA) and DDBJ (Japan). Sample IDs link to the full alignment in AWS S3. MD5 checksums can be found here. For batch downloads, a text file of URLs to BAM files is here.
